# User Structures

## User

| Field        | Type     | Description                |
| ------------ | -------- | -------------------------- |
| id           | String   | The unique ID for the user |
| username     | String   | The user's chosen username |
| email        | String   | The user's email           |
| password     | String   |                            |
| phone_number | String   |                            |
| display_name | String   |                            |
| avatar       | String   |                            |
| banner       | String   |                            |
| birthday     | Date     |                            |
| bio          | String   |                            |
| location     | String   |                            |
| presence     | Presence |                            |
| verified     | Boolean  |                            |
| verifiedAt   | Date     |                            |
| accent_color | String   |                            |
| createdAt    | Date     |                            |
| updatedAt    | Date     |                            |
